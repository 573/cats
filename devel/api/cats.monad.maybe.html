cats.monad.maybe

The Maybe Monad.

from-maybe

(from-maybe mv)(from-maybe mv default)

Return inner value from maybe monad. Accepts an optional default value.

Let see some examples:

(from-maybe (just 1))
;=> 1
(from-maybe (just 1) 42)
;=> 1
(from-maybe (nothing))
;=> nil
(from-maybe (nothing) 42)
;=> 42
