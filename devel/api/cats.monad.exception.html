cats.monad.exception

The Try Monad.

The Try type represents a computation that may either result in an exception, or return a successfully computed value. It’s similar to, but semantically different from the Either type.

failure

(failure e)(failure e message)

A failure type constructor.

If a provided parameter is an exceptio, it wraps it in a Failure instance and return it. But if a provided parameter is arbitrary data, it tries create an exception from it using clojure ex-info function.

Take care that ex-info function in clojurescript differs a little bit from clojure.
